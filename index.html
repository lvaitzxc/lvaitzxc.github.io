深度学习的数据如何导入？

发表于 2020-09-06

深度学习最基本的一步操作是数据集的预处理。如果你研究的问题是算法的基准测试，需要得出自己算法进步的地方，可利用tensorflow已有的数据集（包括自然语言处理、图像识别、目标检测等各类型经典数据集），只需要利用合适地读取即可。

但当遇到的是一个新问题、新场景，需要自己采集数据的情况下，应该熟悉如何对数据进行进行预处理。常见的数据类型有以下类型：

  • 图像
  • 文本
  • 表格（CSV)
  • npy
  • 视频
  • 声音

图像数据的导入

深度学习运用最为成熟就是图像领域和自然语言处理，所以图像数据集的导入也很简单。这里介绍基于tensorflow框架下的图像数据导入，根据tensorflow 2.0的官方API,它提出的解决方案是采用keras.preprocessing对本机上的数据进行加载。

数据集的建立

batch_size=24
img_width=180
img_height=180
根据你图像的这些数据可调

然后利用keras.preprocessing.image_dataset_from_direction()导入,分成训练集和测试集

train_ds = tf.keras.preprocessing.image_dataset_from_directory(
data_dir,
validation_split=0.2,
subset="training",
seed=123,
image_size=(img_height, img_width),
batch_size=batch_size)

val_ds = tf.keras.preprocessing.image_dataset_from_directory(
data_dir,
validation_split=0.2,
subset="validation",
seed=123,
image_size=(img_height, img_width),
batch_size=batch_size)

利用这种方法导入数据需要注意的是，本地保存的图像数据根据类别存放在不同的文件夹中，文件夹的命名以类名名称命名。除此之后，tensorflow对导入的数据的存取操作流程，采用了cache和prefetch进行优化，防止数据存取阻塞，使得网络的训练效率更高。其中关键的两个函数就是.cache函数和.prefetch函数,.cache实现预存，.prefetch实现预取。至此，就完成了图像数据集的导入

AUTOTUNE = tf.data.experimental.AUTOTUNE  #随机数，可以直接设定为自己指定的值比如 1000
train_ds = train_ds.cache().prefetch(buffer_size=AUTOTUNE)
val_ds = val_ds.cache().prefetch(buffer_size=AUTOTUNE)

CSV数据集的导入

csv数据之前的常见导入方法是通过pandas库的read_csv函数直接导入，但是tensorflow提供了它们模块化的导入流程，值得学习一下。

CSV数据集的建立

tensorflow的做法不是将数据通过pandas导入，而是利用tf.data.experimental.make_csv_dataset模块获得初始的训练集和测试集。Tensorflow的文档以泰坦尼克号乘客为数据集，根据乘客的年纪、性别、是否单独旅行等条件判断其是否生还。这是一个基于有监督的分类学习。首先将数据下载保存到本地路径file_path.

LABEL_COLUMN = 'survived' #这是标签类，存在两种情况，死亡和生还分别以0、1表示
LABELS = [0, 1]

def get_dataset(file_path, **kwargs):
  #tensoflow的方案是通过tf.data.experimentak.make_csv_dataset导入本地的数据的csv文件
    dataset = tf.data.experimental.make_csv_dataset(
  file_path,  #csv文件的本地路径
  batch_size=5, # Artificially small to make examples easier to show.
  label_name=LABEL_COLUMN, #标签列
  na_value="?",  #无效值以问号代替
  num_epochs=1,  
  ignore_errors=True, 
  **kwargs)
  return dataset

raw_train_data = get_dataset(train_file_path)
raw_test_data = get_dataset(test_file_path)

CSV数据的预处理流程

早期对数据的处理方法都是利用一些数据处理库，最著名的就是sklearn库了，在模型之外进行数据的预处理然后再将处理好的数据导入模型中进行训练。这样的做法的缺点再于，输入网络的数据不是原始的采集数据，而是每次对原始数据要进行处理，这样的话存在的问题就是保存的模型并不包含这部分预处理部分，这对模型的跨平台使用极不友好，没更换一个平台都需要写数据处理模块的代码。tensorflow的做法是把数据预处理的部分，作为网络的一个层级，放在模型最开始的地方，这样保存的模型就包含了数据预处理的部分。对于CSV格式数据如何处理呢？首先知道，CSV格式的数据一般分为两类，一种是数值型数据，还有一种是属性类数据（比如文本、时间、经纬度等包含字符的数据）。

对于数值型数据

 #将表中所有数值型数据打包成numberic列
class PackNumericFeatures(object):
     def __init__(self, names):
     self.names = names

    def __call__(self, features, labels):
    numeric_features = [features.pop(name) for name in self.names]
    numeric_features = [tf.cast(feat, tf.float32) for feat in numeric_features]
    numeric_features = tf.stack(numeric_features, axis=-1)
    features['numeric'] = numeric_features
        return features, labels

泰坦尼克号乘客数据集中，为数值型的类包含 age,n_siblings_spouses,parch, fare，打包后原先的列名都不存在，统一为numeric列。

NUMERIC_FEATURES = ['age','n_siblings_spouses','parch', 'fare']
 packed_train_data = raw_train_data.map(
PackNumericFeatures(NUMERIC_FEATURES))
packed_test_data = raw_test_data.map(
PackNumericFeatures(NUMERIC_FEATURES))


对于数值型数据，对数据进行标准化是基本操作，标准化的意义就是让数据的分布在小区间一般为[0,1]或者[-1,1],这有利于网络的训练过程。

import pandas as pd
desc = pd.read_csv(train_file_path)[NUMERIC_FEATURES].describe()
MEAN = np.array(desc.T['mean']) #均值
STD = np.array(desc.T['std'])  #方差

def normalize_numeric_data(data, mean, std):
# Center the data
  return (data-mean)/std
#通过funtools.partial可以完成数值型数据的标准化
 normalizer = functools.partial(normalize_numeric_data, mean=MEAN, std=STD)

 numeric_column = tf.feature_column.numeric_column('numeric', normalizer_fn=normalizer, shape=[len(NUMERIC_FEATURES)])
 numeric_columns = [numeric_column]
 #这里通过DenseFeatures使数值型数据的预处理部分作为了网络模型的一层
 numeric_layer = tf.keras.layers.DenseFeatures(numeric_columns)

对于属性型数据

以泰坦尼克号乘客数据集来看，属于属性型数据的列包含sex、class、deck、embark_town、alone,同样的思路就是把他们聚合成只有categorical一列。属性值的处理其实就是对其进行编码，有点类似独热编码的过程。

CATEGORIES = {
'sex': ['male', 'female'],
'class' : ['First', 'Second', 'Third'],
'deck' : ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J'],
'embark_town' : ['Cherbourg', 'Southhampton', 'Queenstown'],
'alone' : ['y', 'n']
}
categorical_columns = []  #这里开始进行编码
for feature, vocab in CATEGORIES.items():
  cat_col = tf.feature_column.categorical_column_with_vocabulary_list(
    key=feature, vocabulary_list=vocab)
 categorical_columns.append(tf.feature_column.indicator_column(cat_col))
#这里通过DenseFeatures使属性型数据的预处理部分作为了网络模型的一层
categorical_layer = tf.keras.layers.DenseFeatures(categorical_columns)

合并两类数据

preprocessing_layer = tf.keras.layers.DenseFeatures(categorical_columns+numeric_columns)

以上完成了对CSV数据的导入，最终它的保存方式很简洁，以下是一个样本的举例，前20个数字是属性型数字后面4个是数值型数据
